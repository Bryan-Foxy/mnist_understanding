# CNN Model
In this section, we will implement a Convolutional Neural Network (CNN) model to process images using a convolutional neural network architecture before feeding it into a Deep Neural Network (DNN). Convolutional Neural Networks have the advantage of considering spatial configurations, making them suitable for handling more complex problems on images.

After establishing the convolutional neural network, we will create a saliency map and GradCAM (Gradient-weighted Class Activation Mapping) to visualize how our CNN distinguishes between a 6 and a 9, and so on.
