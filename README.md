# Modèle CNN
Dans cette partie, on mettra en place un modèle CNN pour traiter les images avec un reseaux de neurones convolutionnelles afin de le mettre dans un DNN.
Les reseaux de neurones convolutionnels ont l'avantage de prendre en compte les configurations spatiales et de ce fait peuvent gerer des problemes plus complexes sur des images.
Apres l'etablissment du reseaux de neurones convolutifs, nous allons dresser une carte de saillance et de gradCAM qui permettra de voir comment notre CNN fait pour differencier un 6 d'un 9, etc...
